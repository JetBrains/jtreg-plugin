# Jtreg plugin for IntelliJ IDEA

<!-- Plugin description -->
Enhance the development workflow with support for <a href="https://openjdk.org/jtreg/">JTReg tests</a> directly within the IDE.
<ul>
  <li><strong>Create JTReg Tests</strong><br>
      Create and manage JTReg tests directly within the IDE.
  </li>
  <li><strong>Customizable Settings</strong><br>
      Configure global JTReg settings and override them per run/debug configuration. Define test selection options to match your specific requirements.
  </li>
  <li><strong>Seamless Navigation</strong><br>
      Browse and manage test directories effortlessly. Generate run/debug configurations with a single action.
  </li>
  <li><strong>Run and Debug Configurations</strong><br>
      Execute individual tests or entire directories through specialized run and debug configurations.
  </li>
  <li><strong>Integrated Test Results</strong><br>
      View comprehensive test results in the dedicated <strong>Test Runner</strong> tab of the Run tool window for efficient review and analysis.
  </li>
</ul>

<!-- Plugin description end -->
